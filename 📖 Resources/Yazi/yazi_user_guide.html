üí° Pro Tip: Shell Wrapper Function

Add this to your ~/.bashrc or ~/.zshrc to enable directory persistence:

function yy() {
    local tmp="$(mktemp -t "yazi-cwd.XXXXXX")"
    yazi "$@" --cwd-file="$tmp"
    if cwd="$(cat -- "$tmp")" && [ -n "$cwd" ] && [ "$cwd" != "$PWD" ]; then
        cd -- "$cwd"
    fi
    rm -f -- "$tmp"
}

After adding, run: source ~/.bashrc (or ~/.zshrc)

First Steps

Launch Yazi

Type yazi in your terminal and press Enter. You'll see a three-panel interface.

Navigate with Arrow Keys

Use ‚Üë ‚Üì to move up/down, ‚Üê ‚Üí to go into/out of directories.

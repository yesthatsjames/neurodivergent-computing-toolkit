## üß† Neurodivergent Design Principles

All tools in this collection follow these principles:

### 1. **Visual-First Design**
- Spatial layouts over text lists
- Color coding for quick categorization
- Icons and emojis as visual anchors
- Preview/context always visible

### 2. **Reduced Cognitive Load**
- Smart defaults (less decision-making)
- Consistent patterns (muscle memory)
- External state tracking (less working memory)
- Visual feedback (no guessing)

### 3. **Flow-State Protection**
- Minimal context switching
- Keyboard-centric (no mouse hunting)
- Fast operations (no waiting)
- Tab/workspace support (parallel tasks)

### 4. **Executive Function Support**
- Automation over repetition
- Templates and shortcuts
- Clear, reversible actions
- Recovery mechanisms

### 5. **Customization**
- Adapt to YOUR brain, not "standard" brains
- Configuration over convention
- Personal optimization encouraged
- No "one size fits all"

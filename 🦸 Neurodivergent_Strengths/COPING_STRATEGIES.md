### Focus Management
- **Pomodoro Technique** - 25 min work, 5 min break
- **Body doubling** - Work alongside others (in person or virtual)
- **Time blocking** - Schedule specific tasks in calendar
- **Two-minute rule** - If it takes <2 min, do it now
- **"Eat the frog"** - Hardest task first thing

- Website blockers (Cold Turkey, Freedom)
- Task managers with notifications
- Visual timers
- Lo-fi music / white noise

| Challenge | Solution |
|-----------|----------|
| Starting tasks | 5-minute commitment only |
| Finishing projects | Break into tiny milestones |
| Forgetting things | External brain (notes, calendar) |
| Hyperfocus crashes | Set alarms for breaks |
| Decision fatigue | Automate routine choices |

## ğŸ“š Dyslexia Strategies

### Reading Code
- **Syntax highlighting** - Use high-contrast themes
- **Font choice** - OpenDyslexic, Comic Sans, or sans-serif
- **Text-to-speech** - Let the computer read to you
- **Screen rulers** - Guides to track lines
- **Increased spacing** - Line height 1.5-2.0

### Writing Code
- **Voice coding** - Dictate code with Talon Voice
- **Autocomplete** - Copilot, IntelliSense
- **Linters** - Catch typos automatically
- **Pair programming** - Real-time review
- **Templates** - Reduce from-scratch writing

### Variable Naming
- Use descriptive names (less memory load)
- Keep a project glossary
- Use IDE search instead of memorizing
- Comment liberally

### Sensory Management
- **Lighting** - Adjust blue light, use lamps instead of overhead
- **Sound** - Headphones, white noise, quiet rooms
- **Clothing** - Comfortable, tag-free
- **Temperature** - Control your workspace temp
- **Breaks** - Step away when overstimulated

### Communication
- **Written > verbal** - Request emails for important info
- **Meeting agendas** - Know topics in advance
- **Processing time** - "Let me think about that"
- **Direct language** - Be clear, expect clarity
- **Video off** - Use chat or voice when cameras drain you

### Routine & Structure
- Same workspace, same time
- Predictable schedule when possible
- Rituals for starting/ending work
- Clear task definitions
- Documented processes
